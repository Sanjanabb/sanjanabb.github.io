HTML Tags

Note: Every tag has opening and closing angle braces.
Also every tag has opening and closing tag

Tag name Usage
html Start and end of html programming tag
body This is the start of the body of html program
title This talks about the main title of the program
br This is used to break a paragraph
p This is used to write a paragraph
ul This is called unordered list and used to write a list of points with bullets
ol This is called ordered list and used to write points with numbers
li To write a signle pint inside ordered or unordered list we use 'li'tag
h1, h2, h3 and so on This is used to write headings. h1 is the first heading and each heading is bolded and the size of the heading is in decending order from h1,h2, h3...
em This tag turns the text in italic style
strong This tag is used to bold the text
img This is used to insert an image into the program
a This is used to insert a link into the progam
table This is used to insert a table into the program
section This is used to represent a section of code
id To give styling to a code lines and background we use this tag
class To give styling to the entire block of code e use this tag
